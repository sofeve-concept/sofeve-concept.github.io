Qu'est-ce que l'accessibilité numérique ?

À l'origine : le web...

Le World Wide Web a été inventé dans les années 1990 par Tim Berners-Lee qui souhaitait alors mettre le Web et ses services à la disposition de tous les individus, quel que soit :

  • leur matériel ou logiciel,
  • leur infrastructure réseau,
  • leur langue maternelle,
  • leur culture,
  • leur localisation géographique,
  • ou leurs aptitudes physiques ou mentales.

En 1994, afin d'assurer un web ouvert, libre et transparent pour tous, Tim Berners-Lee crée le W3C (World Wide Web Consortium), un organisme chargé de standardiser les technologies du web.

... Et l'accessibilité numérique

En 1999, afin de préserver le souhait de son inventeur, un guide d'accessibilité international, le WCAG (Web Content Accessibility Guidelines), paraît. Ce guide édite des recommandations visant à rendre le contenu Web plus accessible, principalement pour les personnes handicapées, mais également pour tous les utilisateurs quel que soit leur moyen d'accès.

L'accessibilité numérique est donc un ensemble de bonnes pratiques et de règles dont l'objectif est de rendre les supports numériques (web, applications mobiles, documents bureautiques) en respectant ces 4 principes :

  1. Compréhensible : Les informations et l'utilisation de l'interface utilisateur doivent être compréhensibles ;
  2. Perceptible :  L'information et les composants de l'interface utilisateur doivent être présentés à l'utilisateur de façon à ce qu'il puisse les percevoir ;
  3. Utilisable simplement :  Les composants de l'interface utilisateur et de navigation doivent être fonctionnels et utilisables simplement ;
  4. Robuste : Optimiser la compatibilité avec les agents utilisateurs actuels et futurs, y compris les technologies d'assistance.
Le pouvoir du Web est son universalité. Qu’il soit accessible par n’importe qui quel que soit son handicap est un de ses aspects essentiels.

Tim Berners-Lee
