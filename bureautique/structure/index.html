Structure du document

La structure du document est importante pour guider l'utilisateur dans la lecture du document.

Titraille

L'utilisation des titres permet de naviguer plus facilement de titre en titre dans le document. De plus, cela permet de créer automatiquement une table des matières.

Liste à puces et liste numérotée

Les logiciels de synthèse vocale disposent d’une option permettant de parcourir les listes de manière efficiente, annonçant même le nombre d’items de la liste.

Colonnes

Pour présenter du texte sur plusieurs colonnes, utiliser la fonctionnalité « Colonnes » prévue à cet effet.

Citations

Le saviez-vous ? Dans tous les logiciels de traitement de texte, il existe un style spécifique pour identifier les citations.

Langue

Le fait de déclarer une langue principale pour le document permet de bénéficier des outils de corrections orthographiques et grammaticales. cela permet aussi aux logiciels de synthèse vocale de parler "dans le bonne langue".
