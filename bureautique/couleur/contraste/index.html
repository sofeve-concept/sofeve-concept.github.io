Contraste de couleur

Comment s'assurer que le contraste entre le fond et le premier plan est suffisant et comment le corriger.

Dans cette rubrique

  • Règles
  • Pourquoi ?
  • Comment ?

Règles

Le contraste entre le texte et la couleur de fond doit être suffisamment élevée. Le ratio de couleur minimum est de 4.5.:1

Pourquoi ?

L’alternative textuelle est lue par les logiciels de synthèse vocale et permet donc de donner de l’information.

Comment ?

Un contraste suffisamment élevé permet aux personnes ayant des troubles de la vision de bien percevoir le texte.

Pour connaître le code couleur d’une police de caractère,

  1. choisir dans le menu « Format - Caractères » (ou cliquer-droit sur le style dans le volet latéral, puis « Modifier » puis choisir l’onglet « Effets de caractères »),
  2. cliquer sur la liste déroulante « Couleur de police » puis sur « Couleurs personnalisées ».
  3. noter les chiffres indiqués « Rouge – Vert – Bleu » (RVB) dans la zone qui s’affiche.

Pour connaître le code couleur d’un arrière-plan (N.B. par défaut le fond d’un document Writer est blanc, code RVB : 255,255,255),

  1. choisir dans la fenêtre des propriétés de l’élément l’onglet « Arrière-Plan »
  2. et noter le code RVB.

Aller ensuite sur le site Contrast Finder et renseigner les codes trouvés de cette manière : - rgb (chiffre_pour_le_rouge, chiffre_pour_le_vert, chiffre_pour_le_bleu),

  • Cliquer sur le bouton « Trouver des couleurs valides » et suivre les instructions.

Pour en savoir plus sur les contrastes de couleurs, consulter l’article de la société Tanaguru sur ce sujet.
